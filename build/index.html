You can customize the function or simply randomize the multipliers used below.
The function is rendered by simply rendering 60,000 dots on a canvas.
The position of the dots is determined using a non-linear equation.

A great number of effects can be achieved simply by changing one multiplier.
In fact, changing one multiplier over time is how the function is animated.
Different effects can also be achieved by using the Math.sin/Math.cos functions.
